Welcome to the Model Hub!
=========================

This is your centralised directory for Machine Learning deployment stacks and inference endpoints. If you are using (or planning to use)
AI in your application or service, we can offer various things:

- **Infrastructure Autotuning**: For any given model, the Model Hub benchmarks all kind of deployment services and deployment stacks,
  finding the best infrastructure for your use case, whether you are aiming for high-throughput, low cost or low latency.

- **Streamlined Model Access**: Every model has a unified API associated, which means that you can switch between providers seamlessly.
  Additionaly, you only need a Unify API Key to access all these endpoints. Only **one account** to use them all.

- **Impartial Arbiter**: We have no vested interest in you using one provider or another, we deploy models everywhere and objectively measure
  their performance to route your requests to the deployment stack that better fits your needs, no need to deal with all the available services
  yourself.

- **No vendor lock-in**: Thanks to the unfied API, we are able to provide fallback models and allow you to switch between providers at any point.
  Don't worry about restructuring your application or refactoring your code. We deal with the intricacies to save you from the pain. If you are no
  longer happy with a provider or it turns out to be unreliable, you are free to move to a different one.

Getting Started
---------------

The best way to try the Model Hub yourself is to get an API Key and go through the tutorial in Make your First Request. Once you are familiar with the
interface and have access to the endpoints, then the Web Interface is the place to go. There, you can search for models and see the differences between
endpoints through the benchmarks before running your queries.

Uploading your own models
-------------------------

You will very soon be able to upload your own models to the Model Hub, both publicly and privately. We will do all the benchmarking and deployment for
you so that you only have to worry about your application and use the best deployment stack at all times. We are testing this feature internally at the
moment, but if you really want to deploy your models in the hub, feel free to send us an email to :code:`modelhub@unify.ai`!
