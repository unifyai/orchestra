Welcome to Unify!
=================

Unify is your centralized platform for LLM endpoints. If you’re using or planning to use an LLM in your application or service, Unify lets you:

* **🔑 Use any endpoint with one key**: You only need a Unify API Key to access the latest LLMs through any endpoint provider. Each model has a unfied API that enables seamless provider switching without needing to refactor.

* **🚀 Route to the best endpoint**: Each prompt is assessed separately and sent to the endpoint that will (a) return the best response, and (b) yield the best performance for a target metric of your choice, including high-throughput, low cost or low latency.

* **⚖️ Benchmark endpoint performance**: Our live, dynamic benchmarks let you objectively compare endpoints speed, latency and costs; to make informed decisions when selecting the model or provider that fits your needs.

Getting Started
---------------

The best way to try the Hub yourself is to `get an API Key <https://unify.ai/docs/hub/home/getting_access.html>`_ and go through the
tutorial in `Make your First Request <https://unify.ai/docs/hub/home/make_your_first_request.html>`_. Once you are familiar with the
interface and have access to the endpoints, then the `Web Interface <https://unify.ai/hub>`_ is the place to go. Now, you can pick any
model and see the differences between its endpoints before running your queries!

Uploading your own models
-------------------------

You will very soon be able to upload your own models to the Hub, both publicly and privately. We will do all the benchmarking and deployment for
you so that you only have to worry about your application and use the best deployment stack at all times. We are testing this feature internally at the
moment, but if you really want to deploy your models in the hub, feel free to send us an email to :code:`hub@unify.ai`!
